# Brand Assets

Drop files here before starting any design session. Claude checks this folder automatically before writing frontend code.

## Folder Structure

| Folder | What goes here |
|--------|---------------|
| `logos/` | SVG, PNG, WebP logos — all variants (light, dark, icon-only, full wordmark) |
| `colors/` | Color palette files — JSON, CSS custom properties, Figma export, or a plain text hex list |
| `fonts/` | Custom font files (`.woff2`, `.ttf`) or a text file with Google Fonts URLs |
| `images/` | Hero images, product shots, lifestyle photography, textures, patterns |
| `docs/` | Brand guidelines PDF, style guide, tone-of-voice doc, any written reference |

## Priority Rules (enforced by CLAUDE.md)
- **Logo present** → use it, never placeholder
- **Color palette defined** → use exact values, never invent brand colors
- **Style guide present** → match it precisely, do not improve
